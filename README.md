# Stepik Database Schema

Этот репозиторий содержит SQL-скрипт для создания и заполнения учебной базы данных, имитирующей платформу **Stepik**. Здесь есть пользователи, курсы, прогресс обучения, сертификаты, комментарии, отзывы и многое другое.

## Структура базы данных
Ключевые таблицы, с которыми можно поработать:
*   **`users`** - пользователи платформы.
*   **`courses`** - курсы.
*   **`user_courses`** - связь пользователей и курсов.
*   **`progresses`** - прогресс по шагам.
*   **`certificates`** - сертификаты.
*   **`comments`** и **`course_reviews`** - комментарии и отзывы.
*   И ещё 8 таблиц для полного погружения.

## Назначение
База данных создана для тренировки SQL-запросов любого уровня сложности:
*   Простые `SELECT` и `JOIN`.
*   Агрегатные функции и группировки (`GROUP BY`).
*   Вложенные и коррелированные подзапросы.
*   Оконные функции (`ROW_NUMBER`, `RANK`, `SUM OVER` и др.).
*   Общие табличные выражения (`CTE`).
*   Работа с датами.

## Пример запроса
Для наглядности, вот как можно найти пользователей с максимальным количеством пройденных шагов:
```sql
SELECT u.full_name, COUNT(p.step_id) AS steps_passed
FROM users u
JOIN progresses p ON u.id = p.user_id
WHERE p.is_passed = TRUE
GROUP BY u.id
ORDER BY steps_passed DESC;
